👤 신규 사용자 승인

신규 회원가입 시 사용자는 PENDING(대기) 상태로 등록됩니다. 관리자가 승인해야 정상 사용이 가능합니다.

승인 절차

더보기 → 사용자 관리
  1. 사용자 관리 페이지로 이동합니다.
  2. 역할 필터에서 "PENDING"을 선택합니다.
  3. 대기 중인 사용자 목록을 확인합니다.
  4. 승인할 사용자의 "역할 변경" 버튼을 클릭합니다.
  5. 적절한 역할을 선택합니다:
    • TEAM_LEADER: TBM 작성, 안전점검, 결재 요청 가능
    • APPROVER: 결재 승인 권한 (임원)
    • ADMIN: 모든 관리 권한
  6. 현장과 팀을 지정합니다.
  7. "저장"을 클릭하여 승인을 완료합니다.
PENDING 사용자는 공지사항 조회만 가능합니다. 교육, TBM, 점검 기능을 사용하려면 반드시 승인이 필요합니다.

👥 사용자 관리

역할 변경

📌 상황: 팀원이 팀장으로 승진했을 때
  1. 사용자 관리 페이지에서 해당 사용자를 찾습니다.
  2. "역할 변경" 버튼을 클릭합니다.
  3. 역할을 TEAM_LEADER로 변경합니다.
  4. 저장합니다.

팀 변경

📌 상황: 사용자가 다른 팀으로 이동했을 때
  1. 사용자 관리 페이지에서 해당 사용자를 찾습니다.
  2. "팀 변경" 버튼을 클릭합니다.
  3. 새로운 팀을 선택합니다.
  4. 저장합니다.

🔐 비밀번호 초기화

사용자가 비밀번호를 분실했을 때 관리자가 초기화할 수 있습니다.

업무 흐름도

비밀번호 분실 신고
↓
사용자 관리 페이지
↓
해당 사용자 찾기
↓
"비밀번호 리셋" 클릭
↓
임시 비밀번호 생성
↓
사용자에게 전달

비밀번호 초기화 절차

  1. 사용자 관리 페이지로 이동합니다.
  2. 검색창에서 해당 사용자를 찾습니다.
  3. "비밀번호 리셋" 버튼을 클릭합니다.
  4. 생성된 임시 비밀번호가 표시됩니다.
  5. 임시 비밀번호를 사용자에게 안전하게 전달합니다.
  6. 사용자에게 로그인 후 비밀번호 변경을 안내합니다.
사용자가 직접 "비밀번호 찾기" 기능을 사용할 수도 있습니다. 로그인 페이지의 "아이디/비밀번호를 잊으셨나요?" 링크를 통해 이메일로 재설정 링크를 받을 수 있습니다.
임시 비밀번호는 화면에 한 번만 표시됩니다. 반드시 기록하거나 즉시 사용자에게 전달하세요.

🏢 팀/팀원 관리

팀원 관리 접근 방법

더보기 → 팀원 관리

팀원 추가

📌 상황: 새로운 팀원을 등록해야 할 때

시스템 계정이 없는 팀원도 TBM 서명을 위해 등록할 수 있습니다.

  1. 팀원 관리 페이지로 이동합니다.
  2. 팀을 선택합니다.
  3. "팀원 추가" 버튼을 클릭합니다.
  4. 팀원 정보를 입력합니다:
    • 이름 (필수)
    • 직급 (선택)
    • 연락처 (선택)
  5. 저장합니다.

팀장/결재자 지정

📌 상황: 팀의 팀장이나 결재자를 변경해야 할 때
  1. 팀원 관리 페이지에서 해당 팀을 선택합니다.
  2. 팀장으로 지정할 팀원 옆의 "팀장 지정"을 클릭합니다.
  3. 결재자 지정이 필요한 경우, 결재자(임원)를 선택합니다.
팀장은 해당 팀의 TBM 작성과 월별 보고서 결재 요청 권한을 갖습니다. 결재자는 해당 팀의 월별 보고서를 승인할 수 있습니다.

📚 교육 과정 관리

접근 방법

더보기 → 교육 관리

새 교육 과정 등록

  1. 교육 관리 페이지로 이동합니다.
  2. "새 교육 추가" 버튼을 클릭합니다.
  3. 교육 정보를 입력합니다:
    • 과정명: 교육 제목
    • 과정 유형: 작업장안전, 위험예방, TBM활동 등
    • 예상 소요시간: 분 단위
    • 교육 영상: YouTube URL 또는 파일 업로드
    • 교육 자료: PDF, 문서 등 첨부
  4. 저장합니다.

평가 문제 관리

📌 상황: 교육에 평가 문제를 추가해야 할 때
  1. 교육 관리에서 해당 과정을 선택합니다.
  2. "문제 관리" 버튼을 클릭합니다.
  3. "문제 추가"를 클릭합니다.
  4. 문제 정보를 입력합니다:
    • 문제 내용
    • 4개의 보기
    • 정답 선택
  5. 필요한 만큼 문제를 추가합니다.
  6. 저장합니다.
평가는 80% 이상 정답 시 합격입니다. 최소 5문제 이상 등록을 권장합니다.

교육 현황 모니터링

더보기 → 교육 현황

팀별, 사용자별 교육 이수 현황을 확인할 수 있습니다.

✅ 체크리스트 템플릿 관리

TBM 일일점검에서 사용하는 체크리스트 항목을 관리합니다.

접근 방법

더보기 → 체크리스트 관리

체크리스트 항목 추가

  1. 체크리스트 관리 페이지로 이동합니다.
  2. 팀을 선택합니다.
  3. 카테고리를 선택하거나 새로 만듭니다.
  4. "항목 추가"를 클릭합니다.
  5. 점검 항목 내용을 입력합니다.
  6. 저장합니다.

항목 순서 변경

드래그 앤 드롭으로 항목 순서를 변경할 수 있습니다.

카테고리별로 항목을 구분하면 TBM 작성 시 더 체계적으로 점검할 수 있습니다. 예: "작업 전 점검사항", "안전수칙 전달사항" 등

팀별 장비 등록

더보기 → 팀별 장비 관리
  1. 팀별 장비 관리 페이지로 이동합니다.
  2. 팀을 선택합니다.

📢 공지사항 관리

공지사항 작성

  1. 홈 또는 공지사항 페이지에서 "새 공지 작성"을 클릭합니다.
  2. 공지 정보를 입력합니다:
    • 제목: 공지 제목
    • 카테고리: 일반, 안전, 교육, 기타
    • 내용: 공지 본문
  3. 필요시 파일을 첨부합니다:
    • 이미지 (JPG, PNG)
    • 동영상 (MP4) 또는 YouTube URL
    • 문서 (PDF, HWP, DOCX)
  4. "등록"을 클릭합니다.

공지사항 수정/삭제

공지사항 상세 페이지에서 수정 또는 삭제할 수 있습니다.

삭제된 공지사항은 복구할 수 없습니다. 삭제 전 반드시 확인하세요.

🔍 문제 해결

자주 발생하는 문제

❓ 사용자가 로그인할 수 없다고 합니다
  1. 계정 상태 확인: 사용자 관리에서 해당 사용자가 PENDING 상태인지 확인합니다.
  2. 비밀번호 확인: 비밀번호가 맞는지 확인하고, 필요시 초기화합니다.
  3. 브라우저 캐시: 사용자에게 브라우저 캐시 삭제 후 재시도를 안내합니다.
❓ TBM을 작성했는데 보이지 않습니다
  1. 팀 확인: 올바른 팀으로 TBM이 작성되었는지 확인합니다.
  2. 날짜 확인: 조회하는 날짜 범위가 맞는지 확인합니다.
  3. 저장 확인: TBM 작성 후 저장 버튼을 눌렀는지 확인합니다.
❓ 이메일이 발송되지 않습니다
  1. SMTP 설정 확인: 이메일 설정에서 SMTP 정보가 올바른지 확인합니다.
  2. 테스트 메일 발송: 테스트 메일 버튼으로 발송 테스트를 합니다.
  3. 수신 이메일 확인: 사용자의 이메일 주소가 올바른지 확인합니다.
❓ 교육 진행률이 저장되지 않습니다
  1. 네트워크 확인: 인터넷 연결 상태를 확인합니다.
  2. 영상 재생: 영상을 재생한 상태에서 잠시 기다립니다 (5초마다 자동 저장).
  3. 브라우저 확인: 다른 브라우저로 시도합니다.
