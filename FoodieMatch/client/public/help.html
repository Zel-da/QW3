사용자 도움말

안전관리 플랫폼 이용 안내

1. 로그인/회원가입

회원가입

  1. 로그인 화면에서 "회원가입" 클릭
  2. 사용자명, 이름, 이메일, 비밀번호 입력
  3. "가입 신청" 클릭
  4. 관리자 승인 후 로그인 가능 (1~3일 소요)

로그인

  1. 사용자명과 비밀번호 입력
  2. "로그인" 버튼 클릭
주의: 5회 이상 비밀번호 오류 시 15분간 로그인이 제한됩니다.

비밀번호 찾기

  1. "아이디/비밀번호를 잊으셨나요?" 클릭
  2. "비밀번호 찾기" 탭 선택
  3. 가입 시 등록한 이메일 입력
  4. 이메일로 받은 링크로 새 비밀번호 설정

2. 홈 대시보드

로그인 후 첫 화면입니다. 주요 현황을 한눈에 볼 수 있습니다.

표시 정보

  • 금일 TBM 현황: 오늘 TBM 작성 여부
  • 이번 달 TBM 작성률: 영업일 대비 작성 비율
  • 안전점검 상태: 이번 달 점검 완료 여부
  • 교육 이수 현황: 등록된 교육 진행률

3. 공지사항

회사 공지사항을 확인할 수 있습니다.

공지 확인

  1. 홈 화면의 공지사항 영역 또는 메뉴에서 접근
  2. 제목 클릭하여 상세 내용 확인
  3. 첨부파일이 있으면 다운로드 가능
새 공지가 있으면 로그인 시 팝업으로 알려줍니다. "확인" 클릭 후 다시 표시되지 않습니다.

4. 안전교육

온라인 안전교육을 수강하고 평가를 통과해야 이수됩니다.

교육 수강 방법

  1. "안전교육" 메뉴 클릭
  2. 수강할 교육 과정 선택
  3. "교육 시작" 버튼 클릭
  4. 동영상 시청 (100% 완료 필요)
  5. "평가 시작" 버튼으로 평가 응시
  6. 80% 이상 정답 시 이수 완료
동영상은 5초마다 자동 저장됩니다. 중간에 종료해도 이어서 시청할 수 있습니다.

평가 기준

기준내용
합격 점수80% 이상
재시험불합격 시 무제한 재응시 가능

5. TBM 일지

매일 작업 전 안전점검 회의(Tool Box Meeting)를 기록합니다. TEAM_LEADER 권한 필요

TBM 작성 방법

  1. "TBM" 메뉴 클릭
  2. "새 TBM 작성" 버튼 클릭
  3. 날짜, 작업 내용, 위험요소, 안전대책 입력
  4. 참석자 선택 후 서명 받기
  5. 체크리스트 항목 확인
  6. 팀장 서명 후 "저장" 클릭

참석자 서명

  1. 참석자 목록에서 팀원 선택
  2. "서명 받기" 버튼 클릭
  3. 해당 팀원이 터치/마우스로 서명
  4. "확인" 클릭하여 저장
  5. 부재 시: "부재 처리" 클릭 후 사유 선택 (연차, 반차, 출장 등)

6. 월별 안전점검

매월 4일에 실시하는 장비 점검입니다. TEAM_LEADER 권한 필요

점검 방법

  1. "안전점검" 메뉴 클릭
  2. 점검할 월 선택
  3. 장비별 점검 상태 선택 (양호/불량)
  4. 불량 시 비고란에 내용 기재
  5. 장비 사진 촬영/업로드
  6. "저장" 클릭
점검일(매월 4일)이 지나면 해당 월 점검이 불가능합니다. 기한 내 완료하세요.

7. 결재

월별 보고서 결재를 요청하고 확인합니다.

결재 요청 (팀장)

  1. "월별 보고서" 메뉴에서 보고서 작성 완료
  2. "결재 요청" 버튼 클릭
  3. 결재자가 승인/반려 처리

결재 현황 확인

  1. "결재 이력" 메뉴 클릭
  2. 요청한 결재 또는 받은 결재 확인
  3. 상태: 대기중 / 승인 / 반려

결재 처리 (결재자)

  1. "받은 결재" 탭에서 대기 중인 결재 선택
  2. 내용 검토
  3. 서명 입력 후 "승인" 또는 사유 입력 후 "반려"

8. 프로필 설정

개인정보 및 비밀번호를 변경할 수 있습니다.

비밀번호 변경

  1. 우측 상단 프로필 아이콘 클릭
  2. "비밀번호 변경" 클릭
  3. 현재 비밀번호, 새 비밀번호 입력
  4. "변경" 클릭
비밀번호는 8자 이상, 영문+숫자+특수문자 포함해야 합니다.

9. 자주 묻는 질문

Q. 교육 진행률이 저장되지 않아요

A. 동영상을 재생한 상태에서 5초 이상 기다리면 자동 저장됩니다. 네트워크 연결 상태를 확인하고, 다른 브라우저(Chrome 권장)로 시도해 보세요.

Q. TBM에서 팀원이 안 보여요

A. 관리자에게 해당 팀원을 팀에 추가해 달라고 요청하세요.

Q. 로그인이 안 돼요

A.

  • 신규 가입 후라면 관리자 승인을 기다리세요
  • 5회 이상 실패 시 15분 후 재시도하세요
  • 비밀번호를 잊었다면 "비밀번호 찾기"를 이용하세요

Q. 화면이 깨지거나 느려요

A. 브라우저 캐시를 삭제하고 새로고침(Ctrl+Shift+R)하세요. Chrome 또는 Edge 최신 버전 사용을 권장합니다.
