사용자 도움말

로그인

로그인 방법

  1. 시스템 URL 접속: https://tbm-0nu9.onrender.com
  2. 아이디와 비밀번호 입력
  3. "로그인" 버튼 클릭
로그인 실패 5회 시 15분간 로그인이 제한됩니다.

비밀번호를 잊었을 때

  1. 로그인 화면에서 "비밀번호를 잊으셨나요?" 클릭
  2. 가입 시 등록한 이메일 입력
  3. 이메일로 받은 링크를 클릭하여 새 비밀번호 설정

회원가입 후 로그인이 안 될 때

회원가입 후 관리자 승인이 필요합니다. 승인 전까지는 일부 기능만 사용 가능합니다.

홈 대시보드

로그인하면 바로 보이는 메인 화면입니다. 오늘 해야 할 일과 현황을 한눈에 확인할 수 있습니다.

항목 내용
공지사항 읽지 않은 공지 개수
교육현황 이번 달 완료/전체 교육 수
TBM현황 이번 달 작성일/영업일 수
안전점검 이번 달 점검 완료 여부
결재대기 처리해야 할 결재 건수

공지사항

공지사항 확인하기

  1. 좌측 메뉴에서 "공지사항" 클릭
  2. 목록에서 확인할 공지 선택
  3. 첨부파일이 있으면 다운로드 가능

공지 팝업

  • 새로운 중요 공지가 있으면 로그인 시 팝업으로 표시됩니다
  • 이미 읽은 공지는 다음 로그인 시 팝업이 표시되지 않습니다
  • "오늘 하루 보지 않기"를 선택하면 당일은 팝업이 표시되지 않습니다

안전교육

교육 수강 방법

  1. 좌측 메뉴에서 "교육" 클릭
  2. 수강할 교육 과정 선택
  3. "학습 시작하기" 또는 "이어서 학습하기" 클릭
  4. 동영상을 100% 시청하면 "평가 시작" 버튼 활성화
진행률 자동 저장
동영상 시청 중 5초마다 자동으로 진행률이 저장됩니다.
중간에 나가도 다음에 이어서 시청할 수 있습니다.

평가 응시

  1. 동영상 100% 시청 완료
  2. "평가 시작" 버튼 클릭
  3. 각 문제에서 정답을 선택하고 "다음" 클릭
  4. 모든 문제 답변 후 "제출" 클릭
항목 내용
문제 유형 4지선다형
합격 기준 80% 이상 정답
재응시 횟수 제한 없음 (불합격 시 바로 재응시 가능)

수료증

평가 합격 후 "수료증 보기"를 클릭하면 PDF 수료증을 다운로드할 수 있습니다.

TBM 일일점검

매일 작업 전 안전점검을 기록하는 기능입니다. 팀장(TEAM_LEADER) 권한이 필요합니다.

TBM 작성하기

  1. 좌측 메뉴에서 "TBM 일일점검" 클릭
  2. 날짜 확인 (기본: 오늘)
  3. 담당 팀/라인 선택
  4. 체크리스트 각 항목에 상태 선택:
    • 양호 - 이상 없음
    • 보통 - 주의 필요
    • 불량 - 조치 필요
  5. 특이사항 입력 (필요시)
  6. 참석 팀원 서명 수집
  7. "저장" 버튼 클릭

팀원 서명 받기

  1. 서명 영역에서 팀원 이름 클릭
  2. 서명 패드에 손가락/마우스로 서명
  3. "확인" 클릭하여 저장
  4. 부재 시: "부재 처리" 클릭 후 사유 선택 (연차, 반차, 출장 등)
수정 가능 기간
당일 작성한 TBM만 수정 가능합니다. 다음 날부터는 조회만 가능합니다.

월별 안전점검

매월 4일에 실시하는 장비 점검입니다. 팀장(TEAM_LEADER) 권한이 필요합니다.

점검 방법

  1. 좌측 메뉴에서 "안전점검" 클릭
  2. 점검할 월과 팀 선택
  3. 점검 대상 장비 선택 (지게차, 크레인, 소화기 등)
  4. "사진 추가" 버튼으로 장비 사진 촬영/업로드
  5. 필요시 특이사항 입력
  6. 모든 장비 점검 후 "점검 완료" 클릭
필수 사진
각 장비별로 지정된 수의 사진을 첨부해야 합니다. 미첨부 시 점검 완료가 불가능합니다.

결재

결재 요청하기 (팀장)

  1. 좌측 메뉴에서 "월별 보고서" 클릭
  2. 해당 월의 보고서 내용 확인 (TBM 현황, 점검 결과)
  3. "결재 요청하기" 버튼 클릭
  4. 결재자(임원) 확인 후 요청

결재 처리하기 (임원)

  1. 좌측 메뉴에서 "결재" 클릭
  2. "받은 결재" 탭에서 대기 중인 건 선택
  3. 보고서 내용 검토
  4. 승인: 서명 입력 후 "승인" 클릭
  5. 반려: 사유 입력 후 "반려" 클릭

결재 상태

상태 설명
대기중 결재 요청 후 처리 대기
승인 결재 완료
반려 수정 후 재요청 필요

내 정보 관리

비밀번호 변경

  1. 우측 상단 프로필 아이콘 클릭
  2. "내 정보" 선택
  3. "비밀번호 변경" 클릭
  4. 현재 비밀번호와 새 비밀번호 입력
  5. "저장" 클릭

이메일 변경

  1. 우측 상단 프로필 아이콘 클릭
  2. "내 정보" 선택
  3. 이메일 수정
  4. "저장" 클릭

자주 묻는 질문

Q. 교육 진행률이 저장되지 않아요

A. 동영상을 재생한 상태에서 5초 이상 기다리면 자동 저장됩니다. 네트워크 연결 상태를 확인하고, 다른 브라우저(Chrome 권장)로 시도해 보세요.

Q. TBM에서 팀원이 안 보여요

A. 관리자에게 해당 팀원을 팀에 추가해 달라고 요청하세요.

Q. 로그인이 안 돼요

A.

  • 신규 가입 후라면 관리자 승인을 기다리세요
  • 5회 이상 실패 시 15분 후 재시도하세요
  • 비밀번호를 잊었다면 "비밀번호 찾기"를 이용하세요

Q. 화면이 깨지거나 느려요

A. 브라우저 캐시를 삭제하고 새로고침(Ctrl+Shift+R)하세요. Chrome 또는 Edge 최신 버전 사용을 권장합니다.
