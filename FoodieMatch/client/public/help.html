사용자 도움말

수산세보틱스 안전관리 플랫폼

1. 로그인/회원가입

회원가입

  1. 로그인 화면에서 "회원가입" 클릭
  2. 사용자명, 이름, 이메일, 비밀번호 입력
  3. "가입 신청" 클릭
  4. 관리자 승인 후 로그인 가능 (1~3일 소요)

로그인

  1. 사용자명과 비밀번호 입력
  2. "로그인" 버튼 클릭
주의: 5회 이상 비밀번호 오류 시 15분간 로그인이 제한됩니다.

비밀번호 찾기

  1. "아이디/비밀번호를 잊으셨나요?" 클릭
  2. "비밀번호 찾기" 탭 선택
  3. 가입 시 등록한 이메일 입력
  4. 이메일로 받은 링크로 새 비밀번호 설정

2. 홈 대시보드

로그인 후 첫 화면입니다. 주요 현황을 한눈에 볼 수 있습니다.

표시 정보

  • 공지사항: 읽지 않은 새 공지 건수
  • TBM 현황: 이번 달 제출 현황 (예: 5/20일 제출)
  • 안전교육: 이수 완료 과정 수 (예: 2/5 완료)
  • 안전점검: 이번 달 점검 완료 여부 및 점검일 안내
안전점검 메뉴는 팀장(TEAM_LEADER), 임원(EXECUTIVE_LEADER), 관리자(ADMIN) 역할만 표시됩니다.

추가 기능

  • 최근 공지: 최근 등록된 공지사항 5건 표시
  • 새 공지 팝업: 읽지 않은 중요 공지가 있으면 로그인 시 팝업으로 알림
  • 최근 활동: 교육, TBM, 공지 등 최근 활동 이력

3. 공지사항

회사 공지사항을 확인할 수 있습니다.

공지 확인

  1. 홈 화면의 공지사항 영역 또는 메뉴에서 접근
  2. 제목 클릭하여 상세 내용 확인
  3. 첨부파일이 있으면 다운로드 가능

댓글 기능

공지사항 상세 페이지 하단에서 댓글을 작성하고 소통할 수 있습니다.

  1. 댓글 입력란에 내용 작성
  2. 필요 시 이미지나 파일 첨부 (선택)
  3. "등록" 클릭하여 댓글 게시
  4. 본인이 작성한 댓글은 수정/삭제 가능
새 공지가 있으면 로그인 시 팝업으로 알려줍니다. "확인" 클릭 후 다시 표시되지 않습니다. 댓글 작성 중인 내용은 임시 저장되어 페이지를 새로고침해도 복원됩니다.

4. 안전교육

온라인 안전교육을 수강하고 평가를 통과해야 이수됩니다.

교육 수강 방법

  1. "안전교육" 메뉴 클릭
  2. 수강할 교육 과정 선택
  3. "교육 시작" 버튼 클릭
  4. 동영상 시청 (100% 완료 필요)
  5. "평가 시작" 버튼으로 평가 응시
  6. 80% 이상 정답 시 이수 완료
동영상은 5초마다 자동 저장됩니다. 중간에 종료해도 이어서 시청할 수 있습니다.

평가 기준

기준내용
합격 점수80% 이상
재시험불합격 시 무제한 재응시 가능

5. TBM 일지

매일 작업 전 안전점검 회의(Tool Box Meeting)를 기록합니다. TEAM_LEADER 또는 EXECUTIVE_LEADER 권한 필요

TBM 작성 방법

  1. "TBM" 메뉴 클릭
  2. "새 TBM 작성" 버튼 클릭
  3. 날짜, 작업 내용, 위험요소, 안전대책 입력
  4. 참석자 선택 후 서명 받기
  5. 체크리스트 항목 확인
  6. 팀장 서명 후 "저장" 클릭

참석자 서명

  1. 참석자 목록에서 팀원 선택
  2. "서명 받기" 버튼 클릭
  3. 해당 팀원이 터치/마우스로 서명
  4. "확인" 클릭하여 저장

참석자 부재 처리

부재 팀원은 출석 상태를 변경하여 사유를 기록할 수 있습니다.

음성 녹음 및 자동 변환 (STT)

TBM 회의 내용을 음성으로 녹음하고, 텍스트로 자동 변환할 수 있습니다.

  1. TBM 작성 화면에서 녹음 버튼 클릭
  2. 회의 내용 녹음 (최대 30분)
  3. 일시정지/재개 가능 (중간에 페이지를 닫아도 녹음 데이터 보존)
  4. 녹음 완료 후 "음성 변환(STT)" 클릭 시 텍스트로 자동 변환
  5. 변환된 텍스트가 비고란에 자동 입력
녹음 데이터는 브라우저에 임시 저장되므로 페이지를 닫거나 새로고침해도 유지됩니다. 녹음 중 마이크 권한이 필요합니다.

자동 저장 (임시 저장)

작성 중인 TBM은 자동으로 임시 저장되어, 실수로 페이지를 닫아도 복원됩니다.

  • 작성 중인 내용이 자동으로 브라우저에 임시 저장
  • 같은 날짜/팀의 TBM을 다시 열면 임시 저장 데이터 복원 여부 확인
  • 최종 저장 후 임시 저장 데이터 자동 삭제

6. 월별 안전점검

매월 지정된 점검일에 실시하는 장비 점검입니다. TEAM_LEADER 또는 EXECUTIVE_LEADER 권한 필요

점검 방법

  1. "안전점검" 메뉴 클릭
  2. 점검할 월 선택
  3. 장비별 점검 상태 선택 (양호/불량)
  4. 불량 시 비고란에 내용 기재
  5. 장비 사진 촬영/업로드
  6. "저장" 클릭
주의: 점검일은 현장별로 관리자가 설정합니다 (기본값: 매월 4일). 점검일이 지나면 해당 월 점검이 불가능하므로 기한 내 완료하세요.

7. 결재

월별 보고서 결재를 요청하고 확인합니다.

결재 요청 (팀장)

  1. "월별 보고서" 메뉴에서 보고서 작성 완료
  2. "결재 요청" 버튼 클릭
  3. 결재자가 승인/반려 처리

결재 현황 확인

  1. "결재 이력" 메뉴 클릭
  2. 요청한 결재 또는 받은 결재 확인
  3. 상태: 대기중 / 승인 / 반려

결재 처리 (결재자)

  1. "받은 결재" 탭에서 대기 중인 결재 선택
  2. 내용 검토
  3. 서명 입력 후 "승인" 또는 사유 입력 후 "반려"

8. 교육 이수 현황

내가 수강한 교육 과정의 이수 현황과 진행률을 확인할 수 있습니다.

이용 방법

  1. "교육 이수 현황" 메뉴 클릭
  2. 전체 / 진행 중 / 완료 탭으로 필터링
  3. 각 과정의 진행률, 소요 시간, 평가 점수 확인

표시 항목

항목설명
진행률영상 시청 진행 비율 (%)
소요 시간교육에 투자한 시간
평가 점수평가 응시 결과 (정답/전체)
이수 상태완료 여부 표시
마지막 접속일최근 학습일
검색 기능으로 특정 교육 과정을 빠르게 찾을 수 있으며, 최신순/이름순/진행률순 정렬이 가능합니다.

9. 프로필 설정

개인정보 및 비밀번호를 변경할 수 있습니다.

비밀번호 변경

  1. 우측 상단 프로필 아이콘 클릭
  2. "비밀번호 변경" 클릭
  3. 현재 비밀번호, 새 비밀번호 입력
  4. "변경" 클릭
비밀번호는 8자 이상, 영문+숫자+특수문자 포함해야 합니다.

10. 자주 묻는 질문

Q. 교육 진행률이 저장되지 않아요

A. 동영상을 재생한 상태에서 5초 이상 기다리면 자동 저장됩니다. 네트워크 연결 상태를 확인하고, 다른 브라우저(Chrome 권장)로 시도해 보세요.

Q. TBM에서 팀원이 안 보여요

A. 관리자에게 해당 팀원을 팀에 추가해 달라고 요청하세요.

Q. TBM 작성 중 페이지를 닫아버렸어요

A. TBM은 자동으로 임시 저장됩니다. 같은 날짜의 TBM을 다시 열면 "이전 작성 내용을 복원하시겠습니까?" 확인 메시지가 표시됩니다.

Q. 음성 녹음이 안 돼요

A. 브라우저에서 마이크 권한을 허용해야 합니다. 주소표시줄 왼쪽의 자물쇠 아이콘 클릭 → 마이크 "허용"으로 변경 후 새로고침하세요.

Q. 로그인이 안 돼요

A.

  • 신규 가입 후라면 관리자 승인을 기다리세요
  • 5회 이상 실패 시 15분 후 재시도하세요
  • 비밀번호를 잊었다면 "비밀번호 찾기"를 이용하세요

Q. 화면이 깨지거나 느려요

A. 브라우저 캐시를 삭제하고 새로고침(Ctrl+Shift+R)하세요. Chrome 또는 Edge 최신 버전 사용을 권장합니다.

Q. EXECUTIVE_LEADER 역할이 뭔가요?

A. 임원이면서 팀장 역할을 겸하는 사용자입니다. TBM 작성, 안전점검, 결재 요청뿐 아니라 결재 승인/반려 권한도 함께 갖습니다.
