# 1. 시스템 개요

## 1.1 시스템 목적
수산세보틱스 안전관리 플랫폼은 작업장 안전을 위한 통합 관리 시스템으로, 다음 기능을 제공합니다:
- **안전교육**: 온라인 교육 수강 및 평가
- **TBM 일지**: 매일 작업 전 안전점검 체크리스트
- **안전점검**: 월별 장비 점검 (매월 4일)
- **결재 시스템**: 월별 보고서 임원 결재
- **공지사항**: 안전 관련 공지 및 소통

## 1.2 접속 방법
- **URL**: https://tbm-0nu9.onrender.com
- **지원 브라우저**: Chrome, Edge, Safari (최신 버전)
- **모바일**: 반응형 웹 지원

## 1.3 현장 구분
| 현장코드 | 현장명 |
|---------|--------|
| ASAN | 아산공장 |
| HWASEONG | 화성공장 |

# 3. 회원가입 및 로그인

## 3.1 회원가입 절차

```
┌─────────────────────────────────────────────────────────────┐
│                    회원가입 흐름도                            │
│                                                              │
│  ┌─────────┐    ┌─────────────┐    ┌─────────────┐          │
│  │ 접속    │───▶│ 회원가입    │───▶│ 정보 입력   │          │
│  │ (로그인)│    │ 버튼 클릭   │    │             │          │
│  └─────────┘    └─────────────┘    └──────┬──────┘          │
│                                           │                  │
│                                           ▼                  │
│                                    ┌─────────────┐          │
│                                    │ 가입 완료   │          │
│                                    │ (PENDING)   │          │
│                                    └──────┬──────┘          │
│                                           │                  │
│                                           ▼                  │
│                                    ┌─────────────┐          │
│                                    │ 관리자 승인 │          │
│                                    │ 대기        │          │
│                                    └──────┬──────┘          │
│                                           │                  │
│                                           ▼                  │
│                                    ┌─────────────┐          │
│                                    │ 권한 부여   │          │
│                                    │ 완료        │          │
│                                    └─────────────┘          │
└─────────────────────────────────────────────────────────────┘
```

### 3.1.1 회원가입 입력 항목
| 필드 | 필수 | 설명 |
|------|------|------|
| 아이디 | ✓ | 영문+숫자, 4자 이상 |
| 비밀번호 | ✓ | 8자 이상, 영문+숫자 조합 |
| 비밀번호 확인 | ✓ | 비밀번호와 일치해야 함 |
| 이름 | ✓ | 실명 입력 |
| 이메일 | ○ | 알림 수신용 (선택) |

### 3.1.2 가입 후 상태
- 가입 직후 역할: `PENDING` (대기)
- 접근 가능: 홈, 공지사항만
- 관리자가 역할/팀/현장 지정 후 정상 사용 가능

## 3.2 로그인 절차

### 3.2.1 일반 로그인
1. 로그인 페이지 접속
2. 아이디/비밀번호 입력
3. "로그인" 버튼 클릭
4. 홈 대시보드로 이동

### 3.2.2 로그인 실패 시
- 5회 이상 실패 시 15분간 로그인 제한
- "비밀번호를 잊으셨나요?" → 관리자에게 문의

## 3.3 로그아웃
1. 우측 상단 프로필 아이콘 클릭
2. "로그아웃" 선택
3. 로그인 페이지로 이동

## 6.3 교육 콘텐츠 시청

### 6.3.1 시청 화면
- 동영상/오디오 플레이어
- 진행률 표시 (0~100%)
- 자동 저장 (5초마다)
- 이어서 시청 지원

### 6.3.2 진행률 저장
- 시청 시간 자동 저장
- 중단 후 재접속 시 이어서 시청
- 100% 시청 완료 시 평가 버튼 활성화

## 6.4 평가 (시험)

### 6.4.1 평가 규칙
| 항목 | 내용 |
|------|------|
| 문제 수 | 5~10문제 (과정별 상이) |
| 문제 유형 | 4지선다형 |
| 합격 기준 | 80% 이상 (8/10 문제) |
| 응시 제한 | 최대 3회 |
| 제한 시간 | 없음 |

### 6.4.2 평가 절차
1. 교육 100% 시청 완료
2. "평가 시작" 버튼 클릭
3. 문제 풀이
4. "제출" 버튼 클릭
5. 결과 확인 (즉시)

### 6.4.3 결과 처리
- **합격 (80% 이상)**: 수료증 자동 발급
- **불합격**: 재응시 가능 (최대 3회)
- **3회 불합격**: 관리자에게 문의

## 6.5 수료증

### 6.5.1 수료증 조회
1. "내 수료증" 메뉴 클릭
2. 발급된 수료증 목록 확인
3. 개별 수료증 클릭하여 상세 보기

### 6.5.2 수료증 정보

# 7. TBM 일일점검

## 7.1 TBM 개요

**TBM (Tool Box Meeting)**
- 매일 작업 시작 전 안전점검 회의
- 팀별로 체크리스트 작성
- 팀원 서명 필수

│  └─────────┘    └─────────────┘    └──────┬──────┘          │
│                                           │                  │
│                                           ▼                  │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐      │
│  │ 저장 완료   │◀───│ 팀원 서명   │◀───│ 특이사항    │      │
