## 1. 시스템 소개

### 1.1 안전관리 통합 플랫폼이란?

안전관리 통합 플랫폼은 **작업자 안전교육**과 **작업 전 안전점검(TBM)**을 하나로 통합한 시스템입니다.

**주요 목적**:
- ✅ 온라인 안전교육 제공 및 이수 관리
- ✅ 교육 진행 현황 실시간 모니터링
- ✅ 사용자별 교육 이력 관리
- ✅ 교육 수료증 자동 발급

### 1.2 시스템 구성

```
┌─────────────────────────────────────┐
│   안전관리 통합 플랫폼               │
├─────────────────────────────────────┤
│                                     │
│  📚 안전교육 시스템 (FoodieMatch)   │
│  - 교육 영상 시청                    │
│  - 평가 및 퀴즈                      │
│  - 이수증 발급                       │
│                                     │
│  ✅ TBM 체크리스트                   │
│  - 작업 전 안전점검                  │
│  - 서명 및 기록                      │
│                                     │
└─────────────────────────────────────┘
```

### 1.3 접속 방법

- **주소**: http://localhost:5173 (개발 환경)
- **운영 환경**: 회사 내부망 URL (IT 부서 문의)
- **권장 브라우저**: Chrome, Edge 최신 버전

## 2. 로그인 및 권한 안내

### 2.1 로그인 방법

**1단계**: 시스템 접속 URL을 브라우저에 입력합니다.

**2단계**: 로그인 화면에서 사용자명과 비밀번호를 입력합니다.

`[스크린샷: 로그인 화면]`

**3단계**: "로그인" 버튼을 클릭합니다.

⚠️ **주의사항**:
- 초기 비밀번호는 IT 부서에서 발급받습니다.
- 최초 로그인 후 비밀번호를 변경하시기 바랍니다.

### 2.2 권한 종류 및 설명

시스템에는 5가지 권한이 있으며, 각 권한별로 사용할 수 있는 기능이 다릅니다.

| 권한 | 이름 | 주요 기능 | 대상 |
|------|------|-----------|------|
| 🔴 **관리자** | ADMIN | 모든 기능 사용 가능 | 시스템 관리 담당자 |
| 🟠 **안전팀** | SAFETY_TEAM | 교육 콘텐츠 관리, 현황 모니터링 | 안전보건팀 담당자 |
| 🟡 **팀장** | TEAM_LEADER | 소속 팀원 관리, 교육 현황 확인 | 각 팀 팀장 |
| 🟢 **현장직** | WORKER | 교육 수강, 서명 필수 | 현장 작업자 |
| 🔵 **사무직** | OFFICE_WORKER | 교육 수강, 서명 불필요 | 사무실 근무자 |

### 2.3 권한별 사용 가능 기능

| 기능 | 관리자 | 안전팀 | 팀장 | 현장직 | 사무직 |
|------|--------|--------|------|--------|--------|
| 교육 수강 | ✅ | ✅ | ✅ | ✅ | ✅ |
| 교육 콘텐츠 추가/수정/삭제 | ✅ | ✅ | ❌ | ❌ | ❌ |
| 사용자 권한 변경 | ✅ | ❌ | ❌ | ❌ | ❌ |
| 모든 사용자 관리 | ✅ | ❌ | ❌ | ❌ | ❌ |
| 팀 관리 | ✅ | ❌ | 🟡 (본인 팀만) | ❌ | ❌ |
| 전체 교육 현황 모니터링 | ✅ | ✅ | 🟡 (본인 팀만) | ❌ | ❌ |

💡 **팁**: 안전팀 담당자는 "안전팀" 권한으로 교육 콘텐츠 관리가 가능합니다.

## 4. 사용자 관리

### 4.1 사용자 목록 조회

사용자 관리 화면에서는 모든 사용자의 정보를 확인하고 관리할 수 있습니다.

`[스크린샷: 사용자 관리 화면]`

**접근 방법**:
1. 상단 메뉴에서 **"사용자 관리"** 클릭
2. 사용자 목록이 표시됩니다

**화면 구성**:
- 🔍 **검색창**: 이름 또는 사용자명으로 검색
- 📊 **팀 필터**: 특정 팀의 사용자만 표시
- 🔢 **정렬 옵션**: 이름순, 사용자명순, 권한순
- 📄 **페이지네이션**: 한 페이지에 10명씩 표시

### 4.2 사용자 검색 방법

**1단계**: 검색창에 찾으려는 사용자의 **이름** 또는 **사용자명**을 입력합니다.

**2단계**: 입력과 동시에 자동으로 검색 결과가 표시됩니다.

💡 **팁**:
- "홍길동"을 찾으려면 "홍" 또는 "길동"만 입력해도 찾을 수 있습니다.
- 검색어를 지우면 전체 목록이 다시 표시됩니다.

### 4.3 사용자 권한 변경

⚠️ **중요**: 권한 변경은 신중하게 진행해주세요!

**1단계**: 변경하려는 사용자의 **권한** 드롭다운을 클릭합니다.

**2단계**: 새로운 권한을 선택합니다.
- 관리자
- 안전팀
- 팀장
- 현장직
- 사무직

**3단계**: 자동으로 저장되며, 화면 우측 상단에 "성공" 메시지가 표시됩니다.

📌 **권한 변경 시 주의사항**:
- ✅ **현장직**: 서명이 필수인 작업자
- ✅ **사무직**: 서명이 필요 없는 사무실 근무자
- ⚠️ **잘못 변경 시**: 즉시 다시 변경하거나 IT 부서에 문의

### 4.4 현장 배정

사용자를 특정 현장에 배정할 수 있습니다.

**1단계**: 배정하려는 사용자의 **현장** 드롭다운을 클릭합니다.

**2단계**: 현장을 선택합니다.
- 아산
- 화성

**3단계**: 자동으로 저장됩니다.

### 4.5 사용자 삭제

⚠️ **경고**: 사용자를 삭제하면 해당 사용자의 모든 교육 기록도 함께 삭제됩니다!

**1단계**: 삭제하려는 사용자의 "삭제" 버튼을 클릭합니다.

**2단계**: 확인 메시지가 표시되면 "확인"을 클릭합니다.

**3단계**: 삭제가 완료되며, 목록에서 사라집니다.

📌 **주의사항**:
- 현재 로그인한 관리자 자신은 삭제할 수 없습니다.
- 삭제된 사용자는 복구할 수 없으므로 신중하게 결정해주세요.

## 5. 교육 콘텐츠 관리

### 5.1 교육 콘텐츠 관리 화면

교육 관리 화면에서는 새로운 교육 과정을 추가하거나 기존 과정을 수정/삭제할 수 있습니다.

`[스크린샷: 교육 콘텐츠 관리 화면]`

**접근 방법**:
1. 상단 메뉴에서 **"교육 관리"** 클릭 (관리자 또는 안전팀 권한 필요)
2. 상단: 새 교육 추가 폼
3. 하단: 기존 교육 과정 목록

### 5.2 새 교육 과정 추가

#### 5.2.1 교육 영상 등록 (유튜브)

**1단계**: "교육 제목" 입력란에 교육명을 입력합니다.
- 예시: "지게차 안전 운행 수칙", "고소작업대 안전관리"

**2단계**: "상세 설명" 입력란에 교육 내용 요약을 작성합니다.
- 예시: "지게차 운전 시 반드시 지켜야 할 안전 수칙을 학습합니다."

**3단계**: "유튜브 영상 URL" 입력란에 유튜브 주소를 입력합니다.

💡 **유튜브 URL 찾는 방법**:
```
① 유튜브에서 교육 영상 재생
② 브라우저 주소창의 URL 전체를 복사
   예: https://www.youtube.com/watch?v=ABC123DEF456
③ 그대로 붙여넣기
```

**4단계**: "교육 시간 (분)" 입력란에 영상 길이를 분 단위로 입력합니다.
- 예시: 30분 영상 → "30" 입력

**5단계**: (선택사항) 퀴즈 파일을 업로드하려면 다음 섹션을 참고하세요.

**6단계**: "업로드" 버튼을 클릭합니다.

**7단계**: 화면 우측 상단에 "교육 과정이 생성되었습니다" 메시지가 표시되면 성공입니다.

#### 5.2.2 퀴즈 파일 (CSV) 업로드

교육 과정에 평가 문제를 추가하려면 CSV 파일을 업로드합니다.

**CSV 파일 형식**:
```csv
question,options,correctAnswer
"지게차 운전 시 안전벨트 착용은 필수인가요?","예;아니오;경우에 따라 다름;모르겠음",0
"화물 적재 시 최대 높이는?","2m;3m;4m;제한 없음",1
"후진 시 주의사항은?","경적;후방확인;감시자 배치;모두 해당",3
```

📌 **CSV 파일 작성 규칙**:
1. **첫 번째 줄**: 헤더 (question,options,correctAnswer)
2. **두 번째 줄부터**: 실제 문제
3. **question**: 문제 내용 (큰따옴표로 감싸기)
4. **options**: 보기 목록 (세미콜론`;`으로 구분)
5. **correctAnswer**: 정답 번호 (0부터 시작, 0=첫 번째 보기)

💡 **예시**:
- "예;아니오;경우에 따라 다름;모르겠음" → 4개 보기
- correctAnswer가 0 → "예"가 정답
- correctAnswer가 3 → "모르겠음"이 정답

**업로드 방법**:
**1단계**: 위 형식에 맞게 CSV 파일을 작성합니다 (Excel 사용 가능).

**2단계**: "퀴즈 파일 (CSV)" 버튼을 클릭하고 파일을 선택합니다.

**3단계**: 교육 과정 정보와 함께 "업로드" 버튼을 클릭합니다.

⚠️ **주의사항**:
- CSV 파일은 UTF-8 인코딩으로 저장해야 합니다 (Excel에서 저장 시 주의).
- 정답 번호는 반드시 0부터 시작합니다 (1번 보기 = 0).
- 파일 형식이 잘못되면 업로드가 실패합니다.

### 5.3 교육 과정 수정

**1단계**: 교육 과정 목록에서 수정하려는 과정의 **"수정"** 버튼을 클릭합니다.

**2단계**: 수정 팝업창이 표시되면 내용을 변경합니다.

**3단계**: "저장" 버튼을 클릭하여 변경사항을 저장합니다.

### 5.4 교육 과정 삭제

⚠️ **경고**: 교육 과정을 삭제하면 관련된 모든 사용자의 진행 기록과 평가 결과도 함께 삭제됩니다!

**1단계**: 삭제하려는 과정의 **"삭제"** 버튼을 클릭합니다.

**2단계**: 확인 메시지를 읽고 "확인"을 클릭합니다.

**3단계**: 삭제가 완료되며, 목록에서 사라집니다.

### 5.5 교육 과정 검색 및 정렬

많은 교육 과정이 있을 때 쉽게 찾을 수 있습니다.

**검색**:
- 검색창에 과정명 또는 설명의 일부를 입력하면 자동으로 필터링됩니다.

**페이지 이동**:
- 교육 과정이 10개 이상일 경우, 하단에 페이지 번호가 표시됩니다.
- 번호를 클릭하거나 "이전"/"다음" 버튼으로 이동합니다.

## 6. 팀 관리

### 6.1 팀 관리 화면

팀 관리 화면에서는 팀원을 추가/삭제하고 팀장을 지정할 수 있습니다.

`[스크린샷: 팀 관리 화면]`

**접근 방법**:
1. 상단 메뉴에서 **"팀 관리"** 클릭 (관리자 또는 팀장 권한 필요)
2. 관리자: 모든 팀 관리 가능
3. 팀장: 본인 팀만 관리 가능

### 6.2 팀 선택

**관리자의 경우**:
**1단계**: "팀을 선택하세요..." 드롭다운을 클릭합니다.

**2단계**: 관리하려는 팀을 선택합니다.

**3단계**: 선택한 팀의 팀원 목록이 표시됩니다.

**팀장의 경우**:
- 본인이 팀장으로 지정된 팀만 자동으로 표시됩니다.

### 6.3 팀원 추가

**1단계**: 우측 "팀원 추가" 카드에서 **"사용자를 선택하세요..."** 드롭다운을 클릭합니다.

**2단계**: 팀에 소속되지 않은 사용자 목록이 표시됩니다.

**3단계**: 추가하려는 사용자를 선택합니다.

**4단계**: **"팀에 추가"** 버튼을 클릭합니다.

**5단계**: 화면 우측 상단에 "성공" 메시지가 표시되며, 팀원 목록에 추가됩니다.

### 6.4 팀장 지정

**1단계**: 좌측 팀원 목록에서 팀장으로 지정하려는 사람의 **"팀장 지정"** 버튼을 클릭합니다.

**2단계**: 자동으로 저장되며, 해당 팀원의 행이 강조 표시됩니다.

📌 **주의사항**:
- 한 팀에 팀장은 한 명만 지정할 수 있습니다.
- 새로운 팀장을 지정하면 기존 팀장은 일반 팀원이 됩니다.
- 팀장으로 지정된 사용자는 해당 팀 관리 권한을 갖게 됩니다.

### 6.5 팀원 삭제

**1단계**: 삭제하려는 팀원의 **"삭제"** 버튼을 클릭합니다.

**2단계**: 확인 메시지가 표시되면 "확인"을 클릭합니다.

**3단계**: 해당 사용자가 팀에서 제외되며, "팀에 소속되지 않은 사용자" 목록으로 이동합니다.

### 6.6 팀원 검색

팀원이 많을 때 쉽게 찾을 수 있습니다.

**1단계**: 팀원 목록 상단의 검색창에 **이름** 또는 **아이디**를 입력합니다.

**2단계**: 입력과 동시에 자동으로 검색 결과가 표시됩니다.

## 7. 교육 현황 모니터링

### 7.1 대시보드 화면

대시보드에서는 전체 교육 진행 현황을 한눈에 확인할 수 있습니다.

`[스크린샷: 교육 현황 대시보드]`

**접근 방법**:
1. 로그인 후 메인 화면 (대시보드)
2. 또는 상단 메뉴에서 **"대시보드"** 클릭

### 7.2 교육 진행 과정 요약

화면 상단에 3단계로 진행 상태가 표시됩니다.

```
┌──────────────┬──────────────┬──────────────┐
│   1단계      │   2단계      │   3단계      │
│  교육 선택   │  영상 시청   │  평가 응시   │
│              │              │              │
│  완료 ✓      │  진행 중 ⏱   │  대기 중 ⏳  │
└──────────────┴──────────────┴──────────────┘
```

- ✅ **완료** (녹색): 해당 단계 완료
- 🟠 **진행 중** (주황): 현재 진행 중
- ⚪ **대기 중** (회색): 아직 시작 안 함

**우측에 표시되는 정보**:
- "완료한 과정: 5 / 10" → 전체 10개 중 5개 완료

### 7.3 교육 과정 필터링

교육 과정 목록을 상태별로 확인할 수 있습니다.

**1단계**: 화면 중간의 탭을 클릭하여 필터를 선택합니다.
- **전체**: 모든 과정 표시
- **진행중**: 수료하지 않은 과정만 표시
- **수료**: 완료한 과정만 표시

**2단계**: 선택한 탭의 과정 목록이 표시됩니다.

### 7.4 교육 과정 검색

**1단계**: 검색창에 찾으려는 교육 과정명을 입력합니다.

**2단계**: 입력과 동시에 자동으로 검색 결과가 표시됩니다.

### 7.5 교육 과정 정렬

검색창 우측의 정렬 드롭다운을 사용하여 과정 순서를 변경할 수 있습니다.

- **최신순**: 최근에 추가된 과정부터 표시
- **과정명순**: 가나다 순으로 정렬
- **진행률순**: 진행률이 높은 과정부터 표시

### 7.6 개별 사용자 수강 이력 확인

특정 사용자의 교육 이력을 확인하려면:

**방법 1 - 내 이수 현황**:
1. 상단 메뉴에서 **"내 이수 현황"** 클릭
2. 로그인한 사용자의 모든 교육 이력이 표시됩니다

**방법 2 - 다른 사용자 확인 (관리자/안전팀)**:
1. 사용자 관리 화면에서 해당 사용자 확인
2. (향후 기능: 사용자별 상세 이력 조회)

### 7.7 수강 이력 화면 설명

`[스크린샷: 수강 이력 화면]`

**표시되는 정보**:
- **과정명**: 교육 과정 이름 (클릭 시 해당 과정으로 이동)
- **진행률**: 영상 시청 진행률 (0% ~ 100%)
- **시청 시간**: 실제 시청한 시간
- **평가 점수**: 퀴즈 결과 (예: 8 / 10)
- **수료 여부**: ✅ 또는 ❌
- **마지막 학습일**: 가장 최근에 접속한 날짜

**필터 및 정렬**:
- **전체 / 수강중 / 수료**: 상태별 필터링
- **최근 학습순 / 과정명순 / 진행률순**: 정렬 옵션

## 8. 자주 묻는 질문

### Q1: 사용자가 로그인을 못한다고 하면?

**A**: 다음 사항을 확인해주세요.

1. **사용자명 확인**
   - 사용자 관리 화면에서 해당 사용자의 사용자명이 정확한지 확인
   - 대소문자 구분에 주의

2. **비밀번호 초기화**
   - 관리자가 비밀번호를 재설정할 수 있습니다
   - (향후 기능: 비밀번호 재설정 기능 추가 예정)

3. **권한 확인**
   - 사용자 관리 화면에서 해당 사용자에게 권한이 올바르게 부여되었는지 확인

### Q2: 교육 영상이 재생되지 않는다고 하면?

**A**: 다음을 확인해주세요.

1. **유튜브 URL 확인**
   - 교육 관리 화면에서 해당 과정의 유튜브 URL이 올바른지 확인
   - 유튜브에서 해당 영상이 삭제되거나 비공개로 설정되지 않았는지 확인

2. **네트워크 연결**
   - 사용자의 인터넷 연결 상태 확인
   - 회사 방화벽에서 유튜브를 차단하지 않았는지 IT 부서에 문의

3. **브라우저 업데이트**
   - Chrome 또는 Edge 최신 버전 사용 권장

### Q3: 퀴즈 파일 업로드가 실패하면?

**A**: CSV 파일 형식을 다시 확인해주세요.

**체크리스트**:
- [ ] 첫 줄에 헤더가 있나요? (question,options,correctAnswer)
- [ ] 문제 내용이 큰따옴표로 감싸져 있나요?
- [ ] 보기가 세미콜론(`;`)으로 구분되어 있나요?
- [ ] 정답 번호가 0부터 시작하나요? (0, 1, 2, 3...)
- [ ] 파일이 UTF-8 인코딩으로 저장되었나요?

💡 **해결 방법**:
- Excel에서 저장 시: "CSV UTF-8 (쉼표로 분리)(*.csv)" 형식 선택
- 샘플 파일 참고: `quiz_sample.csv` (프로젝트 폴더에 있음)

### Q4: 교육 진행률이 제대로 저장되지 않으면?

**A**: 다음을 확인해주세요.

1. **영상 시청 시간 확인**
   - 영상을 충분히 시청했는지 확인 (건너뛰기는 인정 안 됨)
   - 진행률은 실시간으로 자동 저장됩니다

2. **브라우저 새로고침**
   - 페이지를 새로고침(F5)하여 최신 상태를 확인

3. **로그아웃 후 재로그인**
   - 세션 문제일 수 있으므로 로그아웃 후 다시 로그인

### Q5: 권한을 잘못 부여했을 때 복구 방법은?

**A**: 즉시 다시 변경하면 됩니다.

**1단계**: 사용자 관리 화면으로 이동

**2단계**: 해당 사용자의 권한 드롭다운에서 올바른 권한을 다시 선택

**3단계**: 자동 저장되며, 즉시 적용됩니다

⚠️ **주의**: 현장직과 사무직 구분은 중요합니다!
- **현장직**: TBM 체크리스트에서 서명 필수
- **사무직**: 서명 불필요

### Q6: 교육 과정을 삭제했는데 복구할 수 있나요?

**A**: 아니오, 삭제된 교육 과정은 복구할 수 없습니다.

- 삭제 시 모든 관련 데이터(진행률, 평가 결과 등)도 함께 삭제됩니다
- 삭제하기 전에 반드시 확인 후 진행해주세요
- 중요한 데이터는 정기적으로 백업하는 것을 권장합니다

### Q7: 한 번에 많은 사용자를 등록할 수 있나요?

**A**: 현재는 IT 부서에 일괄 등록을 요청해야 합니다.

- 데이터베이스에 직접 등록하는 방식입니다
- Excel 파일로 사용자 목록을 전달하면 IT 부서에서 처리합니다
- (향후 기능: Excel 업로드 기능 추가 예정)

### Q8: 교육 현황을 Excel로 다운로드할 수 있나요?

**A**: (향후 기능: 현황 다운로드 기능 추가 예정)

## 9. 문제 해결 가이드

### 9.1 일반적인 오류 해결 방법

#### 오류: "로딩 중..." 화면이 계속 표시됨

**원인**: 서버 연결 문제 또는 네트워크 지연

**해결 방법**:
1. 인터넷 연결 상태 확인
2. 페이지 새로고침 (F5)
3. 브라우저 캐시 삭제 후 재접속
4. 다른 브라우저로 시도
5. 지속되면 IT 부서에 문의

#### 오류: "권한이 없습니다" 메시지 표시

**원인**: 해당 기능에 접근할 권한이 없음

**해결 방법**:
1. 사용자 관리 화면에서 본인 권한 확인
2. 관리자에게 필요한 권한 요청
3. 로그아웃 후 재로그인

#### 오류: 변경사항이 저장되지 않음

**원인**: 네트워크 오류 또는 세션 만료

**해결 방법**:
1. 페이지 새로고침하여 최신 상태 확인
2. 로그아웃 후 재로그인
3. 다시 작업 수행
4. 지속되면 IT 부서에 문의

**주간 점검** (안전팀 담당자):
