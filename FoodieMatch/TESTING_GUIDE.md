### 2. TBM 체크리스트 테스트 (핵심 기능)

#### 2.1 TBM 작성
| # | 테스트 항목 | 테스트 방법 | 예상 결과 | 확인 |
|---|------------|------------|----------|------|
| 2.1.1 | 새 TBM 작성 | TBM 메뉴 → 팀 선택 → 날짜 선택 | 체크리스트 양식 표시 | ☐ |
| 2.1.2 | 체크항목 선택 | 각 항목 양호/불량/해당없음 선택 | 선택값 저장됨 | ☐ |
| 2.1.3 | 불량 선택 시 | 불량 선택 → 조치내용 입력 | 조치내용 저장됨 | ☐ |
| 2.1.4 | 사진 첨부 | 불량 항목에 사진 첨부 | 사진 업로드 및 표시 | ☐ |
| 2.1.5 | 참석자 서명 | 참석자 목록에서 서명 | 서명 이미지 저장 | ☐ |
| 2.1.6 | 저장 | 저장 버튼 클릭 | "저장 완료" 메시지 | ☐ |
| 2.1.7 | 자동 저장 | 30초 대기 | 자동 저장됨 | ☐ |

#### 2.2 TBM 조회/수정
| # | 테스트 항목 | 테스트 방법 | 예상 결과 | 확인 |
|---|------------|------------|----------|------|
| 2.2.1 | 목록 조회 | TBM 목록 메뉴 클릭 | 작성된 TBM 목록 표시 | ☐ |
| 2.2.2 | 상세 보기 | 목록에서 항목 클릭 | 상세 내용 표시 | ☐ |
| 2.2.3 | 수정 | 수정 버튼 → 내용 변경 → 저장 | 변경 내용 저장됨 | ☐ |
| 2.2.4 | 날짜 필터 | 날짜 범위 선택 | 해당 기간 TBM만 표시 | ☐ |
| 2.2.5 | 팀 필터 | 팀 선택 | 해당 팀 TBM만 표시 | ☐ |

### 3. 결재 시스템 테스트

### 4. 안전교육 테스트

| # | 테스트 항목 | 테스트 방법 | 예상 결과 | 확인 |
|---|------------|------------|----------|------|

| 4.5 | 평가 통과 | 80% 이상 정답 | "통과" 메시지 + 수료증 | ☐ |
| 4.6 | 평가 불합격 | 80% 미만 정답 | "불합격" + 재시험 안내 | ☐ |

## 테스트 일정

### 전체 일정 (3주)

| 주차 | 기간 | 내용 | 담당 |
|------|------|------|------|
| 1주차 | 1/6 ~ 1/10 | 기능 테스트 | 안전보건팀 |
| 2주차 | 1/13 ~ 1/17 | 버그 수정 | 개발팀 |
| 3주차 | 1/20 ~ | 결과 미팅 | 전체 |

### 1주차: 기능 테스트 상세

| 일차 | 테스트 항목 | 담당 |
|------|------------|------|
| 월 | 로그인, TBM 작성 | 안전보건팀 |
| 화 | TBM 조회, 결재 | 안전보건팀 |
| 수 | 안전교육, 공지사항 | 안전보건팀 |
| 목 | 안전점검 사진, 월간리포트 | 안전보건팀 |
| 금 | 모바일 테스트, 종합 | 현장 테스트 |

### 2주차: 버그 수정
- 1주차 테스트에서 발견된 버그 수정
- 개선 요청사항 반영
- 수정 완료 후 재테스트

### 3주차: 결과 미팅
- 테스트 결과 보고
- 파일럿 팀 선정
- 전사 도입 일정 협의
